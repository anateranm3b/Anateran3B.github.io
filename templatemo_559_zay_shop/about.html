¿Que es cloud computing?

La computación en la nube es la disponibilidad bajo demanda de los recursos del sistema informático, especialmente el almacenamiento de datos y la potencia informática, sin una gestión activa directa por parte del usuario. Las nubes grandes suelen tener funciones distribuidas en múltiples ubicaciones, cada una de las cuales es un centro de datos.

¿Que es SaaS?

El software como servicio es un modelo de entrega y licencia de software en el que el software se licencia mediante suscripción y está alojado de forma centralizada. SaaS también se conoce como software bajo demanda, software basado en web o software alojado en web.

¿Que es PaaS?

La plataforma como servicio o plataforma de aplicación como servicio o servicio basado en plataforma es una categoría de servicios de computación en la nube que permite a los clientes aprovisionar, crear instancias, ejecutar y gestionar un sistema modular.

¿Que es IaaS?

La infraestructura como servicio es un modelo de servicio de computación en la nube mediante el cual un proveedor de servicios en la nube suministra recursos informáticos. El proveedor de IaaS proporciona almacenamiento, red, servidores y virtualización. Este servicio permite a los usuarios liberarse del mantenimiento de un centro de datos local.

¿Que es virtualizacion?

En informática, virtualización o virtualización es el acto de crear una versión virtual de algo en el mismo nivel de abstracción, incluidas plataformas de hardware de computadora virtual, dispositivos de almacenamiento y recursos de red de computadora.

¿Que es Docker?

Docker es un conjunto de productos de plataforma como servicio que utilizan virtualización a nivel de sistema operativo para entregar software en paquetes llamados contenedores. El servicio tiene niveles gratuitos y premium. El software que aloja los contenedores se llama Docker Engine. Se lanzó por primera vez en 2013 y está desarrollado por Docker, Inc.
